Detailed Description

A quoteState returns a quoted string token from a reader. This state will collect characters until it sees a match to the character that the tokenizer used to switch to this state. For example, if a tokenizer uses a double-quote character to enter this state, then

nextToken() will search for another double-quote until it finds one or finds the end of the reader.
